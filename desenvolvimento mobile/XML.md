## Mudar o Id de um componente
Para mudar o id de um componente basta no XML buscar pela propriedade "Id" e mudar conforme sua necessidade. É uma boa prática colocar sempre uma abreviação de o que aquele componente é, por exemplo: 
- btn_botão_legal
- txt_caixa_de_texto_irada
